# Go Authentication Server

Simple authentication api written in go inspired by [gotrue](https://github.com/netlify/gotrue)

#### Requirements
- Golang

### Usage :
#### Debug build
``` console
user@main:~$ go build
user@main:~$ ./go-auth
```
#### Release build
``` console
user@main:~$ go build
user@main:~$ GIN_MODE=release ./go-auth
```
